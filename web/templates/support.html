    Etcd Backup

    Create a backup of the etcd database from a master node. The backup includes the etcd snapshot and static pod resources needed for cluster recovery.

    Etcd Diagnostics

    Analyze etcd database contents to identify large or excessive objects. Results can be copied to clipboard or saved as a text file.

    Database Overview

    Scan the entire etcd database. These commands run directly against etcd and may take a few minutes on large clusters.

    Drill Down by Resource

    Pick a resource type from the overview results (e.g. secrets, events, configmaps) and investigate further.
